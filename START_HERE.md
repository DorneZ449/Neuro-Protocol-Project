## 🚀 Что делать дальше?

### Шаг 1: Установить PostgreSQL (если еще нет)

**Windows:**
1. Скачай с https://www.postgresql.org/download/windows/
2. Установи с настройками по умолчанию
3. Запомни пароль для пользователя `postgres`

**Создай базу данных:**
```sql
-- Открой pgAdmin или psql и выполни:
CREATE DATABASE clienthub;
```

---

### Шаг 2: Настроить Backend

Открой терминал в папке проекта:

```bash
cd "C:\web\api\Project Neuro-Protocol\clienthub-backend"

# Установи зависимости
npm install

# Создай файл .env
copy .env.example .env

# Отредактируй .env - укажи свой пароль PostgreSQL:
# DATABASE_URL=postgresql://postgres:ТВОЙ_ПАРОЛЬ@localhost:5432/clienthub

# Собери проект
npm run build

# Создай таблицы в БД
npm run db:migrate

# Добавь демо-данные (опционально)
npm run db:seed

# Запусти сервер
npm run dev
```

Если всё правильно, увидишь:
```
🚀 Сервер запущен на порту 5000
```

---

### Шаг 3: Настроить Frontend

Открой **НОВЫЙ** терминал (не закрывай первый!):

# Запусти приложение
npm run dev

Если всё правильно, увидишь:

## 🐛 Если что-то не работает

### Ошибка подключения к БД:
```
Error: connect ECONNREFUSED
```
**Решение:**
- Проверь, что PostgreSQL запущен
- Проверь пароль в `.env`
- Проверь, что база `clienthub` создана

### Порт занят:
```
Error: listen EADDRINUSE: address already in use :::5000
```
**Решение:**
- Измени `PORT=5001` в `.env`
- Обновите URL в `clienthub-frontend/src/api/axios.ts`

### CORS ошибки:
```
Access to XMLHttpRequest has been blocked by CORS policy
```
**Решение:**
- Убедись, что backend запущен на порту 5000
- Проверь, что в `server.ts` есть `app.use(cors())`
