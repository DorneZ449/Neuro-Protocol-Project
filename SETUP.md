# Инструкция по запуску ClientHub

## Шаг 1: Установка PostgreSQL

Если у вас еще не установлен PostgreSQL:

### Windows:
1. Скачайте PostgreSQL с https://www.postgresql.org/download/windows/
2. Установите с настройками по умолчанию
3. Запомните пароль для пользователя postgres

### Создание базы данных:
```sql
-- Откройте pgAdmin или psql и выполните:
CREATE DATABASE clienthub;
```

## Шаг 2: Настройка Backend

1. Откройте терминал в папке `clienthub-backend`

2. Установите зависимости:
```bash
npm install
```

3. Создайте файл `.env`:
```bash
copy .env.example .env
```

4. Отредактируйте `.env` (укажите свой пароль PostgreSQL):
```
PORT=5000
DATABASE_URL=postgresql://postgres:ВАШ_ПАРОЛЬ@localhost:5432/clienthub
JWT_SECRET=my-super-secret-key-2024
NODE_ENV=development
```

5. Запустите миграции (создание таблиц):
```bash
npm run build
npm run db:migrate
```

Вы должны увидеть: ✅ Таблицы успешно созданы

6. Запустите сервер:
```bash
npm run dev
```

Сервер должен запуститься на http://localhost:5000

## Шаг 3: Настройка Frontend

1. Откройте НОВЫЙ терминал в папке `clienthub-frontend`

2. Установите зависимости:
```bash
npm install
```

3. Запустите приложение:
```bash
npm run dev
```

Приложение откроется на http://localhost:5173

## Шаг 4: Тестирование

1. Откройте браузер: http://localhost:5173

2. Зарегистрируйтесь:
   - Нажмите "Нет аккаунта? Зарегистрируйтесь"
   - Введите имя, email, пароль
   - Нажмите "Зарегистрироваться"

3. Создайте клиента:
   - Нажмите "+ Добавить клиента"
   - Заполните форму
   - Нажмите "Создать"

4. Откройте карточку клиента:
   - Кликните на карточку клиента
   - Добавьте заказ
   - Добавьте взаимодействие
   - Добавьте комментарий

## Возможные проблемы

### Ошибка подключения к БД:
- Проверьте, что PostgreSQL запущен
- Проверьте правильность пароля в `.env`
- Проверьте, что база данных `clienthub` создана

### Порт занят:
- Измените PORT в `.env` на другой (например, 5001)
- Обновите URL в `clienthub-frontend/src/api/axios.ts`

### CORS ошибки:
- Убедитесь, что backend запущен на порту 5000
- Проверьте, что в `server.ts` есть `app.use(cors())`
