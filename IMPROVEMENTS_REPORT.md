### ✅ 3. Добавлена пагинация

**Что сделано:**
- Backend возвращает данные с пагинацией:
  - `page` - номер страницы
  - `limit` - количество элементов (по умолчанию 12)
  - `total` - общее количество
  - `totalPages` - количество страниц
- Frontend отображает:
  - Кнопки навигации (Назад/Вперёд)
  - Номера страниц
  - Счётчик "Показано X из Y"
- Пагинация сбрасывается при изменении фильтров

**Оптимизация:**
- Запросы к БД оптимизированы с LIMIT и OFFSET
- Подсчёт общего количества выполняется отдельным запросом
