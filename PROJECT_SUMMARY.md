# ClientHub - Итоговое резюме проекта

## 📋 Общая информация

**Название:** ClientHub - CRM система для учёта клиентов и истории взаимодействия

**Дата создания:** 13 апреля 2026

**Тип проекта:** Веб-приложение (Full-stack)

**Уровень:** Middle

---

## 🎯 Решаемая проблема

Малый и средний бизнес теряет клиентов и деньги из-за:
- Отсутствия единой базы клиентов
- Потери истории взаимодействий
- Невозможности отследить повторные продажи
- Хаоса в работе с клиентами

**Решение:** Простая и понятная CRM-система для управления клиентами, заказами и историей взаимодействия.

## 🚀 Быстрый старт

### 1. Установка зависимостей:
```bash
# Backend
cd clienthub-backend
npm install

# Frontend
cd clienthub-frontend
npm install
```

### 2. Настройка:
```bash
# Backend
cd clienthub-backend
copy .env.example .env
# Отредактировать .env (указать DATABASE_URL)
```

### 3. База данных:
```bash
# Создать БД в PostgreSQL
CREATE DATABASE clienthub;

# Запустить миграции
cd clienthub-backend
npm run build
npm run db:migrate
npm run db:seed  # Опционально: демо-данные
```

### 4. Запуск:
```bash
# Backend (терминал 1)
cd clienthub-backend
npm run dev

# Frontend (терминал 2)
cd clienthub-frontend
npm run dev
```

### 5. Открыть:
- Frontend: http://localhost:5173
- Backend API: http://localhost:5000

### 6. Вход:
- Email: demo@clienthub.com
- Пароль: demo123

1. Открыть карточку клиента
2. Увидеть все заказы, взаимодействия и комментарии
3. Отследить повторные касания
